Description

        
Pagination is a component that displays page numbers. It will not manipulate your data collection. You will have to split your data 
collection into pages yourself. 

       
       
Example Code

       
 <div class="">
<div *ngFor="let item of pagingItems">{item.body}</div>
</div>
<app-pagination-comp [items]="results" [pageSize]="10" [maxPages]="5"   (changePage)="onChangePage($event)"></app-pagination-comp>
      
 
          
Variable Description

        

          
          

            Name
            Type
            Default
            Description
          

          
          
          

              initialPage 
              Number
              1
              Frist page in the list
            

            

                pageSize  
                Number
                10
                Default page size in the list
              

              

                  totalItems  
                  Number
                   
                  Total records are aavaliable on the page
                

                

                    maxPages  
                    Number
                    5
                    Maximum page at once
                  

            
        

          
Methods Description

        

          
          

            Name
            Type
            Default
            Description
          

          
          
          

              onChangePage() 
              function
              
              Trigger when value changes
            

            

                ngOnInit()  
                function
                
                initlize the variable once page loads
              

              

                  setPage() 
                  function
                  
                  Trigger when value changes while move next and previous pages and set the page value
