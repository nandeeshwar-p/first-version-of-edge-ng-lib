Description

            
Pagination is a component that displays page numbers. It will not manipulate your data collection. You will have to split your data 
    collection into pages yourself. 

           
           
Example Code

           
  <app-pagination-comp [items]="results" [pageSize]="10" [maxPages]="8"   (changePage)="onChangePage($event)"></app-pagination-comp> 
           
 
              
Variable Description

            

              
              

                Name
                Type
                Default
                Description
              

              
              
              

                   [items] 
                  object
                  
                  Total avaliable records which needs to be paginated
                

                

                    [pageSize]  
                    Number
                    10
                    Records avaliable per page 
                  

                  

                      [maxPages]  
                      Number
                       
                      Number of pages to be display for pagination
                    

                
            

              
Methods Description

            

              
              

                Method Name
                Type
                parameters
                Description
              

              
              
                

                  (changePage) 
                  function
                  event
                  Callback function Trigger whenever page changes 
                

                

                    ngOnInit()  
                    function
                    void
                    initlize the variable and Methods once page loads
